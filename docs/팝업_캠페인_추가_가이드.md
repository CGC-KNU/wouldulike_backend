# 팝업 캠페인 추가 가이드

앱에서 노출되는 팝업 캠페인을 추가하는 방법입니다.

---

## 1. API 엔드포인트

| 메서드 | 경로 | 설명 |
|--------|------|------|
| GET | `/trends/popup_campaigns/` | 현재 노출 중인 팝업 목록 (활성 + 기간 내) |
| GET | `/trends/popup_campaigns/<id>/` | 특정 팝업 상세 |

---

## 2. 팝업 추가 방법

### 방법 A: Management Command (권장)

```bash
python manage.py add_popup_campaign \
  --title "이벤트 팝업" \
  --image-url "https://example.com/popup.png" \
  --instagram-url "https://instagram.com/account" \
  --start-at "2025-03-01 00:00" \
  --end-at "2025-03-31 23:59"
```

**옵션:**

| 옵션 | 필수 | 설명 |
|------|------|------|
| `--title` | ✅ | 팝업 제목 |
| `--image-url` | ✅ | 팝업 이미지 URL |
| `--instagram-url` | ✅ | 인스타그램 링크 |
| `--start-at` | ✅ | 노출 시작일시 (YYYY-MM-DD 또는 YYYY-MM-DD HH:MM, UTC) |
| `--end-at` | ✅ | 노출 종료일시 |
| `--display-order` | | 표시 순서 (작을수록 먼저, 기본값: 0) |
| `--inactive` | | 비활성 상태로 생성 |

### 방법 B: Django Admin

1. `/admin/trends/popupcampaign/` 접속
2. "팝업 캠페인 추가" 클릭
3. 필드 입력 후 저장

---

## 3. 응답 예시 (GET /trends/popup_campaigns/)

```json
[
  {
    "id": 1,
    "title": "이벤트 팝업",
    "image_url": "https://example.com/popup.png",
    "instagram_url": "https://instagram.com/account",
    "start_at": "2025-03-01T00:00:00Z",
    "end_at": "2025-03-31T23:59:00Z",
    "is_active": true,
    "display_order": 0,
    "created_at": "2025-03-02T12:00:00Z",
    "updated_at": "2025-03-02T12:00:00Z"
  }
]
```

---

## 4. 프론트엔드 연동

- 앱 시작 시 또는 특정 화면 진입 시 `GET /trends/popup_campaigns/` 호출
- 응답이 비어있지 않으면 팝업 모달 표시
- `image_url`로 이미지, `instagram_url`로 링크(버튼 등) 연결
