# 앱 접속 쿠폰 - 월/수 주기 발급

## 개요

- **발급 요일**: 월요일, 수요일만 (MON_WED)
- **만료**: 발급일 기준 3일 (예: 월 10:00 발급 → 목 23:59 KST 만료)
- **월요일**: 술집 아닌 제휴식당 중 1개 식당의 쿠폰 1장
- **수요일**: 술집 제휴식당 중 1개 식당의 쿠폰 1장
- **다른 요일**: MON_WED 발급 없음

## 환경 변수 (각각 독립 온오프)

| 변수 | 기본값 | 설명 |
|------|--------|------|
| APP_OPEN_LEGACY_ENABLED | 1 | 기존 전체 발급 (APP_OPEN_COUPON_TYPE_CODE) |
| APP_OPEN_MON_WED_ENABLED | 0 | 월/수 1장씩 (술집X/술집) |

**둘 다 활성화 시 병행 발급** (전체 + 월/수 1장 동시 발급).

```bash
# 월/수 추가 발급만 활성화 (기존 전체 발급 유지)
APP_OPEN_MON_WED_ENABLED=1

# 기존 전체 발급 끄고 월/수만
APP_OPEN_LEGACY_ENABLED=0
APP_OPEN_MON_WED_ENABLED=1
```

## CouponType / Campaign

| 구분 | CouponType | Campaign |
|------|------------|----------|
| 월요일 | APP_OPEN_MON | APP_OPEN_MON_EVENT |
| 수요일 | APP_OPEN_WED | APP_OPEN_WED_EVENT |

## 만료일 표시 예시

- **발급**: 2026-03-16(월) 10:00 KST
- **만료**: 2026-03-19(목) 23:59:59 KST
- **API 응답**: `"expires_at": "2026-03-19T23:59:59+09:00"`
- **프론트엔드**: "~3/19(목) 23:59까지" 또는 "3일 후 만료"

## 중지

캠페인을 중지하려면:

```python
# Django shell
from coupons.models import Campaign
Campaign.objects.filter(code__in=['APP_OPEN_MON_EVENT', 'APP_OPEN_WED_EVENT']).update(active=False)
```

또는 DB에서 `campaigns` 테이블의 `active` 컬럼을 `false`로 변경.

## 술집 구분 기준

- **술집**: `pub_option='네'` 또는 `pub_option`이 `'네,'`로 시작, 또는 `category='술집'`
- **술집 아닌 식당**: 위 조건에 해당하지 않는 제휴식당
